# desenvolvimento_app_academia
Projeto desenvolvido para praticar Python do básico ao avançado através da criação de um sistema de consultoria online para academia. O objetivo é aplicar cada novo conceito aprendido no Curso em Vídeo, evoluindo o projeto continuamente até transformá-lo em uma aplicação completa.

# 🏋️ Sistema de Consultoria Online para Academia

## 📖 Sobre o Projeto

Este projeto foi criado com o objetivo de colocar em prática os conhecimentos adquiridos durante meus estudos de Python.

Ao invés de desenvolver pequenos exercícios isolados, decidi construir uma aplicação que evoluirá junto com meu aprendizado. Cada novo conteúdo estudado será aplicado neste projeto, tornando-o cada vez mais completo.

O tema escolhido foi uma **Consultoria Online para Academia**, área na qual já possuo experiência profissional como Personal Trainer.
